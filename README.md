# cf-fast-dl

Fast DL for custom fortress stuff

also 

## "How to use your [tools](https://github.com/Teeenoob/cf-fast-dl/tree/main/tools) and what they do?"

* You need this if you wanna use .pl file
  * Requires [Perl](https://www.perl.org/) and knowledge how to use it.


## Maps.pl Tool:

Makes mapcycle.txt file with all the map names from same folder (for simple copy paste)

* Copy / Move it to steam team fortress 2 map folder: `...\Steam\steamapps\common\Team Fortress 2\tf\maps`
  * And run it, idk.

## Workshop.pl tool:

It copies all your subscribed maps into one single folder "maps".

* Copy / Move it to steam team fortress 2 workshop folder: `...\Steam\steamapps\workshop\content\440`.
  * And run it, idk.
